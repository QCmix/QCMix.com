# QCmix

QCmix is a web platform built for the Quad Cities bartending and music community.  
It brings together service industry professionals, venues, musicians, and patrons into a single, locally focused ecosystem.

The platform is designed to:
- Provide professional profiles for bartenders, musicians, and venues
- Support community networking and discovery
- Introduce practical tools for service workers, including an in-progress tip tracking application
- Serve as a foundation for future industry-focused features and services

This project is built using modern, production-grade web technologies and follows Next.js App Router best practices.

# QCmix.com - Next.js Migration (Phase 1A)

A dedicated hub for Quad Cities Bartenders, Venues, and Musicians.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Backend/Auth:** Firebase (Authentication + Firestore)

### Project Status

QCmix is under active development.

Current phase focuses on:

- Authentication and user context

Additional functionality will be introduced incrementally.
